## Roadmap

### Phase 1: Foundation
- [X] Packet capture on specified interface
- [X] Parse Ethernet frames (extract MAC)
- [X] Parse IP headers (extract src/dst IP)
- [X] Track devices in memory (MAC → IP, packet count)
- [X] CLI output: live device table

### Phase 2: Protocol Analysis
- [X] Parse DNS queries/responses
- [X] Extract TLS SNI (server names from HTTPS)
- [X] Track domains per device
- [ ] Parse ARP for device discovery

### Phase 3: Persistence
- [X] SQLite schema for devices, events, DNS logs
- [X] Write events to database
- [ ] Query historical data

### Phase 4: Threat Detection
- [ ] Port scan detection
- [ ] Known bad IP/domain blocklist
- [ ] New device alerts
- [ ] Anomaly detection (baseline → deviation)

### Phase 5: Web Dashboard
- [ ] REST API (actix)
- [ ] Device list endpoint
- [ ] Device detail endpoint
- [ ] WebSocket for live updates

### Phase 6: Visualization
- [ ] Basic web UI
- [ ] Device connection graph
- [ ] Traffic over time charts
- [ ] Alert feed

### Phase 7: Deployment
- [ ] Cross-compile for ARM
- [ ] deploy.sh script
- [ ] systemd service file
- [ ] GitHub Actions CI/CD
